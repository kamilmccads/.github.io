UTM Link Generator

For Direct Deals, use the following format: direct-deal-[website name]. For example: ?utm_source=direct-deal-bloomberg.

For CTA and Text Links, use the following format: cta-[button name], text-link-[link name]. For example: &utm_medium=cta-open-an-account, &utm_medium=text-link-open-a-live-account.

The campaign name needs to include the country, language, and license. In some specific cases, indicate if it is for prospecting or retargeting. For example: &utm_campaign=forex-2405-prospecting-uk-en-fca.
